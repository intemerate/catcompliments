# compliments

a simple website that gives you random compliments with a cute cat that follows your mouse

## stuff

- random compliment generator
- animated cat cursor follower
- random cat images in the background

## credits

- cat sprites by [@crgimenes](https://github.com/crgimenes/neko)
- cat images from [cataas.com](https://cataas.com)

## how to run

just open `index.html` and thats mostly it
